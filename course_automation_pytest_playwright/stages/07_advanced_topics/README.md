# 07 — Продвинутые темы

Темы, которые можно изучать дальше:

- Параллельный запуск тестов (`pytest-xdist`) и управление ресурсами
- Тестовые данные: фабрики, fixtures, Faker
- Интеграция с базой данных, мокирование внешних API
- Visual regression testing (сравнение скриншотов)
- Тесты на мобильных разрешениях и cross-browser тестирование
- Мониторинг тестов и оповещения (Slack, Email, Sentry)

Практические задания:

1. Настройте `pytest-xdist` и запустите тесты в 4 потока.
2. Реализуйте fixture, которая поднимает тестовую БД в Docker (Postgres) и выполняет миграции.
3. Настройте отправку отчётов в Slack (через webhook) после завершения pipeline.
